Digit Recognition with MNIST, Keras and Tensorflow

David Gallagher

Project Specs

Train a Neural Network(NN) using Keras and Tensorflow.
The NN will use the MNIST dataset to train and recognise digits 0-9. We then use that trained NN to predict the number passed to it.
Build a client and server application using Flask.
The Flask server will receive an image from the client which has been drawn by the user on a canvas and use the NN to predict what digit is in the image.
